# Getting Started
<br/>
## Installation

Start by installing the DSSSP npm package and add it to your dependencies.

```bash
npm install dsssp
```

## Usage

Then, import the components you need and start using them in your project.

```tsx
import { FrequencyResponseGraph, FilterCurve, FilterPoint } from 'dsssp';
```
