# Getting Started
<br/>
## Installation

Start by installing the DSSSP npm package and add it to your dependencies.

```bash
npm install dsssp
```

## Imports

Then, import the components you need:
```tsx
import {
  FrequencyResponseGraph,
  CompositeCurve,
  FilterCurve,
  FilterPoint,
  FilterGradient,
  FilterIcon
} from 'dsssp'
```

Import the icon font for FilterIcons, FilterPoints, and custom UI elements:

```tsx
import 'dsssp/font'
```

For TypeScript, also import the relevant types:
```tsx
import {
  type GraphTheme,
  type GraphScale,
  type GraphFilter
} from 'dsssp'
```

Declare some [filters](?path=/docs/foundations-audio-filters--docs) to use in your first app.

```tsx
const filters: GraphFilter[] = [
  { freq: 400, gain: 6, q: 1, type: 'PEAK' },
  { freq: 600, gain: -8, q: 3, type: 'PEAK' },
  { freq: 1200, gain: 2, q: 0.7, type: 'HIGHSHELF2' }
]
```

## First application

Finally, create a React app that includes a [FrequencyResponseGraph](?path=/docs/container-frequencyresponsegraph--docs), a few [FilterCurves](?path=/docs/components-filtercurve--docs) and [CompositeCurve](?path=/docs/components-compositecurve--docs) to render the frequency response.

```tsx
const App = () => {
  return (
    <FrequencyResponseGraph width={800} height={480}>
      <FilterCurve filter={filters[0]} color="#FF9966" />
      <FilterCurve filter={filters[1]} color="#6699FF" />
      <FilterCurve filter={filters[2]} color="#66FF66" />
      <CompositeCurve filters={filters} />
    </FrequencyResponseGraph>
  )
}

export default App
```
