## Filter Icons

The library provides an extensive SVG canvas to visualize audio filters in their natural habitat, but various user interfaces also require more schematic representations of filters, outside of the graph. These icons can be used for buttons, menus, or other UI elements.

## Usage

The `FilterIcon` component can be used in two ways: using the `type` property with the filter name or passing the filter object itself as the `filter` prop. Setting a positive or negative `gain` value (either as a separate prop or inside the filter object) can also change the icon's appearance, flipping the direction of the curve.

<div style={{ padding: '12px', display: 'inline-block', background: '#000', border: '1px solid #444',  borderRadius: '4px', marginRight: '20px' }}>
  <PeakIcon type="PEAK" gain={-6} size={38} color='#3399FF'/>
</div>
<div style={{ padding: '12px', display: 'inline-block', background: '#000', border: '1px solid #444',  borderRadius: '4px' }}>
  <HighShelfIcon type="HIGHSHELF2" gain={6} size={38} color='#FF9933'/>
</div>


```tsx
<FilterIcon color="#3399FF" size={38} type="PEAK" gain={-6} />
<FilterIcon color="#FF9933" size={38} type="HIGHSHELF2" gain={6} />
// are the same as
<FilterIcon color="#3399FF" size={38} filter={{ type: 'PEAK', gain: -6, freq: 500, q: 1 }} />
<FilterIcon color="#FF9933" size={38} filter={{ type: 'HIGHSHELF2', gain: 6, freq: 500, q: 1 }} />
```
Among the unified icon component, there are also individual components for each filter type: `PeakIcon`, `NotchIcon`, `LowShelfIcon`, `HighShelfIcon`, `LowPassIcon`, `HighPassIcon`, `BandPassIcon`, `BypassIcon`, `GainIcon`.
