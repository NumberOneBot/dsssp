## Demo Project &nbsp; [Link &#129125;](https://numberonebot.github.io/dsssp-demo/)

This project showcases a music player with a multi-band equalizer built using the library components. The interface provides real-time interaction with filter parameters through drag-and-drop controls and mouse wheel, with immediate visual feedback on the frequency response curve.

The demo demonstrates how to connect filter adjustments with audio processing, linking the SVG visualization to the Web Audio API.

### Try it yourself

You can study the complete source code in its dedicated [repository](https://github.com/numberonebot/dsssp-demo) or open it on [CodeSandbox](https://codesandbox.io/p/github/numberonebot/dsssp-demo/main).

To run the demo locally:

```bash
git clone https://github.com/numberonebot/dsssp-demo.git
npm install
npm run dev
```
